Odoo Machine and Equipment Rental Management Odoo App

Machine and Equipment Rental Odoo apps general functions which can be utilised for any rental business is quite well spread across many industry so it has many name like Hire industry, Equipment rental, Plant hire, is a service industry providing machinery, equipment and tools of all kinds and sizes for a limited period of time to final users. This odoo apps helps user to rent machine and Equipment products based on initial terms like hours, days, weeks and months and bill according that,user can extend rental order also can replace rental product, also recurring invoice and delivery order will generated based on subscription, An email remainder also send to user and customer for rental order expiry.
Odoo system has been popular across world, we thought module like this can be helpful to many people in hire tenure, rental contract and lease business.

Rental Configuration

In email configuration you need to enter days for send email reminder to client.

In order to add saleable product to rental order enable" Add saleable product in rental order".

Rental Product

Under Sales > Rental > Rental Product menu user can create and configure rental product with Monthly Rental and Replacement Value as shown on following image. User can rent a product based on unique serial number.

Rental Order Menu

Under Sales > Rental > Rental Orders menu user can create rental orders, User can enter Customer Information, Rental Start Date, Rental End Date and Warehouse, define Initial Terms(Months) for rental order, Rental bill of frequency will be used for a recurring invoice generation. Now user can add rental product as well as saleable product to rental order, Based on rental bill of frequency(hours,days, weeks,month) next invoice will be automatically set as per recurring subscription has been generated.

Create Rental Order

Based on initial term rental order start date and end date will be automatically set.

Confirming Rental Order

User confirm rental order by pressing "Confirm Rental" button. it will generate validated invoice and delivery order.

After confirm rental order recurring subscription has been generated and based on date of next invoice has been sent.

Delivery Order created from Rental Order

User can see created two delivery orders for rental order of both rental and saleable products.

Invoice created from Rental Order

User can see created single invoice for both rental and saleable products.

Replace Rental Product

User can replace rental product by clicking "Replace Product" button on rental order.
In replace product wizard existing product is displayed and in other section user have to enter details about same product with different serial number.

In order to replace the products delivery order must be in done state if it is not in done state it will display warning.

After rented product there might be a posibility that problem or damage found on product so in this case we can replace same product with different serial number.

In order to replace the products you have to select same product with different serial number, If product is same it will display warning.

User can see new product is replaced with existing product.

New delivery order will be generated for the replacement, So old product will came in warehouse and new serial number of product will be delivered to customer as rented product and old order is cancelled automatically.

Rental Email Reminder

According to remainder days entered in configuration an email will be sent.

1) Reminder to Customer : Customer get notify and get idea that product has about to expired so they can renew or extend rental order.

2) Reminder to Expired Rental : Sales person get notify that the rental contract has been expired so they can renew or extend rental order.

Reminder To Customer

Reminder Email before selecting days that your rental order is about to expire.

Reminder To Sales Person

Reminder Email after expire your rental order.

Extend Rental Order

We have a feature that rental order can be easily renew or extend using "Extend/Renew Rental" button.

In extend rental order wizard user can select "Initial Terms" as hours, days, week and month and enter based on that "Rental Start Date" will automatically set.

Extended rental date is added to rental order.

Extended rental date will also effected on generated subscription and recurring invoice.

Rental Order Report

User can also print rental order report.

Close Rental Order

When rental period is completed you can close rental order via close rental button.

User can see closed rental order and created new delivery order.

Which use can processed and rental product will arrive in your warehouse.

Rental Order History

user can see rental order history on serial number.
