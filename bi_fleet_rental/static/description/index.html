Complete Rental Solution for Fleet Rental, Machine Rental and Equipment Rental Management Odoo App

This odoo app is complete solution for all kind rental services, it helps to rent vehicle/car/fleet as well as rent Product/Machine and rent Equipment product for daily, weekly, monthly, weekly and hourly period.

Fleet and Machine, Equipment, Rental Odoo apps general functions which can be utilised for any rental business is quite well spread across many industry so it has many name like Hire industry, Equipment rental, Plant hire,Fleet or car Rental is a service industry providing machinery,vehicle, equipment and tools of all kinds and sizes for a limited period of time to final users. This odoo apps helps user to rent a fleet,machine and products,user can extend rental order also can replace rental product, also recurring invoice and delivery order will generated based on subscription.

Features

Invoice will create based on rental order subscription.

Rental Configuration

In configuration, user need to enter days based on that, an email reminder will send to the customer, and enable "Add saleable product in rental order" to rent salable products.

Fleet Product

To rent a fleet user have to select rental product on fleet.

Rental Product

Under Sales > Rental > Rental Product menu user can create and configure rental product with Monthly Rental, Rent per Week, Rent per Day, and Replacement Value as shown on following image. User can rent a product based on unique serial number.

Create Fleet Rental Order

Under Sales > Rental > Rental Orders menu user can create rental orders, User can enter Customer Information, Rental Start Date, Rental End Date and Warehouse, define Initial Terms(Months) for rental order, Rental bill of frequency will be used for a recurring invoice generation. Now user can add rental product as well as saleable product to rental order, Based on rental bill of frequency(days,month) next invoice will be automatically set as per recurring subscription has been generated.

Confirming Fleet Rental Order

User confirm fleet rental order by pressing "Confirm Rental" button. it will generate validated invoice and delivery order.

After confirm fleet rental order recurring subscription has been generated and based on date of next invoice has been sent.

Delivery Order created from Fleet Rental Order

User can see created two delivery orders for fleet rental order of both rental and saleable products.

Invoice created from Fleet Rental Order

User can see created single invoice for both rental and saleable products.

Replace Fleet Rental Product

User can replace rental product by clicking "Replace Product" button on rental order.
In replace product wizard existing product is displayed and in other section user have to enter details about same product with different serial number.

In order to replace the products delivery order must be in done state if it is not in done state it will display warning.

After rented product there might be a possibility that problem or damage found on product so in this case we can replace same product with different serial number.

In order to replace the products you have to select same product with different serial number, If product is same it will display warning.

User can see new product is replaced with existing product.

New delivery order will be generated for the replacement, So old product will come in warehouse and new serial number of product will be delivered to customer as rented product and old order is cancelled automatically.

Rental Email Reminder

According to remainder days entered in configuration an email will be sent.

1) Reminder to Customer : Customer get notify and get idea that product has about to expired so they can renew or extend rental order.

2) Reminder to Expired Rental : Sales person get notify that the rental contract has been expired so they can renew or extend rental order.

Reminder To Customer

Reminder Email before selecting days that your rental order is about to expire.

Reminder To Sales Person

Reminder Email after expire your rental order.

Extend Rental Order

We have a feature that rental order can be easily renew or extend using "Extend/Renew Rental" button.

In extend rental order wizard select date till user wants to extend rental order.

Extended rental date is added to rental order.

Extended rental date will also effected on generating subscription and recurring invoices.

Rental Order Report

User can also print rental order report.

Close Rental Order

When rental period is completed you can close rental order via close rental button.

User can see closed rental order and created new delivery order.

Which use can processed and rental product will arrive in your warehouse.

Rental Order History

User can see rental order history on serial number.
